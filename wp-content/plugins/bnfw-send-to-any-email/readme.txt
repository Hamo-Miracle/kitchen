Send notifications to non-WordPress Users. You can use manually typed email addresses or custom field shortcodes (when used with the [Custom Fields add-on](https://betternotificationsforwp.com/downloads/custom-fields/)) in addition to selecting User Roles or Users in the Send To box as normal.
